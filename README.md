# Akka Streams

This repo is created for practicing and understanding coding with 
[Streams](http://doc.akka.io/docs/akka-stream-and-http-experimental/1.0/scala/stream-index.html).

### Build & Run ###

```sh
$ cd akka-streams
$ sbt ~run
```
